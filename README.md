# 🌍 GeoSense - Discover Places Around You

GeoSense is a location-aware web application that demonstrates how geographic data and modern web technologies can be combined to create interactive, real-world applications.

The project focuses on using browser geolocation and map-based interfaces to display useful information about places and surroundings.

🔗 Live Website: https://geosense-ai.vercel.app/

---

## 🚀 Features

- 📍 Uses browser geolocation to detect location
- 🗺 Interactive map-based interface
- ⚡ Fast and lightweight frontend
- 📱 Fully responsive design
- 🌐 Works directly in the browser
- 🎯 Simple and clean UI

---

## 🧠 Project Purpose

GeoSense is designed to demonstrate how geospatial technologies can be integrated into web applications. It helps developers understand how location-based data can be used to build interactive and practical web tools.

The project showcases:

- Geolocation integration
- Interactive web UI
- Real-world mapping concepts

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Browser Geolocation API
